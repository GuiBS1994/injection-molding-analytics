# Injection Molding Analytics

This project aims to build a data pipeline that integrates real-time injection machine data with production orders to generate automated insights and daily reports.

## Objectives
- Monitor cycle time performance
- Identify deviations from standard cycle
- Automate daily reporting
- Enable future integration with ERP/MRP systems

## Architecture (MVP)
- MySQL (machine data)
- SharePoint / Excel (manual OP control)
- Power BI (data processing)
- Power Automate (report delivery)
